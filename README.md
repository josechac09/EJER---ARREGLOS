Este programa consiste en un sistema de registro de ventas mensuales para una tienda que cuenta con tres departamentos: Ropa, Deportes y Juguetería. La información se almacena en un arreglo bidimensional, donde las filas representan los meses del año y las columnas representan los departamentos. Cada posición del arreglo guarda el monto de ventas correspondiente a un mes y a un departamento específico. El programa cuenta con un método para insertar una venta, el cual permite ingresar un mes, un departamento y un monto, y guarda ese valor en la posición correcta del arreglo. También incluye un método para buscar una venta, que permite consultar el monto registrado para un mes y departamento determinados. Además, posee un método para eliminar una venta, el cual reemplaza el valor almacenado por cero, indicando que no existe una venta registrada. El funcionamiento del programa se realiza mediante un menú interactivo que permite al usuario seleccionar la operación que desea realizar hasta decidir finalizar la ejecución.
